Basic usage

A mobile and touch friendly input spinner component for Bootstrap. It supports the mousewheel and the up/down keys.

Sizes

Wrap with .input-group-sm for small size. And .input-group-lg for large size.

Prefix and Postfix

Place one add-on on either side of an input or both.

Vertical buttons

Place the button vertically.
